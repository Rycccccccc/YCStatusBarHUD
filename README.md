# YCStatusBarHUD
一个简单易用的状态栏指示器框架。。。。。。

## 普通显示
```
objc
 [YCStatusBarHUD showMessage:@"普通文字"];
```
   
## 正在加载
```
objc
    [YCStatusBarHUD showLoading:@"正在加载"];
```
## 加载成功
```
objc
    [YCStatusBarHUD showSuccess:@"成功"]
```
## 加载失败
```
objc
    [YCStatusBarHUD showError:@"失败"];
```
## 隐藏
```
objc
    [YCStatusBarHUD hide];
```
